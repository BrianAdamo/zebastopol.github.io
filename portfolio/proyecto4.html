Introducción a Bootstrap

El objetivo del proyecto fue aprender a utilizar Bootstrap para crear diseños web responsivos de manera más sencilla y rápida.

Se implementaron componentes como navbar, card y grid para estructurar el contenido y mejorar la experiencia del usuario.
