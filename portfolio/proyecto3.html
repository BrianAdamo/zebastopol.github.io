Alineando contenidos con Flexbox

El objetivo del proyecto fue aprender a utilizar Flexbox para alinear y distribuir el espacio entre los elementos de una página web de manera eficiente.

Se implementaron propiedades como display: flex, justify-content y align-items para lograr un diseño responsivo y adaptable.
