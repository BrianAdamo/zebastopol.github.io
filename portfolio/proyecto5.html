Gestión de Proyectos con Git y GitHub

El objetivo del proyecto fue aprender a utilizar Git y GitHub para gestionar el código fuente y colaborar en el desarrollo de software.

Se implementaron prácticas como el uso de ramas, commits y pull requests para facilitar el trabajo en equipo y el control de versiones.
